# Hysteria2 Dashboard

Management panel for **[Hysteria2](https://hysteria.network/)** - a fast proxy protocol with censorship bypass.

## What it does

- **Easy install** - Hysteria2 and the panel deploy with a single command
- **Settings** - manage Hysteria2 configuration
- **Keys** - create and delete access keys
- **Traffic** - overall statistics and per-key breakdown

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/SodiumCXI/hysteria2-dashboard/main/install.sh | bash
```

## Uninstall

Run the same command again
