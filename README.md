# Hysteria2 Dashboard

Панель управления для [Hysteria2](https://hysteria.network/) - быстрого прокси-протокола с обходом блокировок.

## Что умеет

- **Простая установка** - Hysteria2 и панель разворачиваются одной командой
- **Настройки** - управление конфигурацией Hysteria2
- **Ключи** - создание и удаление ключей доступа
- **Трафик** - общая статистика и по каждому ключу отдельно

## Установка

```bash
curl -fsSL https://raw.githubusercontent.com/SodiumCXI/hysteria2-dashboard/main/install.sh | bash
```

## Удаление

Запустите эту же команду повторно
